# marg.ing — портфолио видеомонтажера

`marg.ing` — персональное портфолио видеомонтажера.

Лаконичный сайт для презентации работ, опыта, визуального стиля и контактов специалиста в сфере видео, монтажа и контент-продакшена.

## Что внутри

- главный экран с визуальной подачей бренда;
- блок избранных работ;
- отдельная страница со всеми проектами;
- просмотр проектов через встроенный видеоплеер;
- секция с описанием опыта;
- контактный блок с почтой и Telegram;
- адаптивная верстка для компьютеров и мобильных устройств.

## Технологии

- React
- TypeScript
- Vite
- Tailwind CSS
- Motion
- Lucide React
